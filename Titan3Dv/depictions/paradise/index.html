Description


Paradise brings a new way to give a beautiful touch to your Icon’s, Home Screen, App Library and much, much more. Have you always felt the need to make your own icons and set them easily? Are you tired of waiting for developers to make icons for your requested apps? Is it painful to uninstall multiple apps one by one? Well you’re in luck because Paradise is here and it’s bundled with a lot features, let’s go through them one by one.


No designer experience is required to design your own icons for any apps.


Icon Designing App

Paradise comes with an icon designing app which you can use to create some amazing icons it’s easy to use and fully featured. It comes with 2000+ of SF Symbols for icons, the process is simple and straightforward you can design the icon, import the background or icon image from your photo library, set the icon or background colour, save the icon to your photo library, share them via social media, create a new theme folder, select which theme folder you want save your icons to then you can use an theme engine like SnowBoard or iThemer to apply the icons. Paradise will take care of the rest once you finished design your icon, it will save that icon to theme folder of your choice and name the icon from the app bundleID automatically.


Icon Label Colours

You can set custom colours for each app icon’s label separately. Just swipe up on any app icon or use 3D Touch Menu to Edit With Paradise and choose your custom colours for the selected app label.


Icon Label Background Colours

You can set custom colours for each app icon’s label background separately. Just swipe up on any app icon or use 3D Touch to Edit With Paradise and choose your custom colours for the selected app labels background.


Custom App Names

You can use Paradise to set a custom name for the Home Screen app icons. Just swipe up on any app icon or use 3D Touch to Edit With Paradise and choose your custom title for the selected app label.


Icon Label Colours

You can set custom colour for each App Library folder label separately. Just double tap on any folder in the App Library to open so you can edit with Paradise and choose your custom colour for the selected app label.


Folder Label Colours

You can set custom colours for each App Library folder label separately. Just double tap on any folder in App Library to open to edit with Paradise and choose your custom colour for the selected folder label.


Folder Label Background Colours

You can set a custom background colour for each App Library folder label separately. Just double tap on any folder in the App Library to edit with Paradise and choose your custom background colour for the selected folder label.


Icon Label Title (App Library)

You can set a title for each App Library folder separately. Just double tap on any folder in App Library to open so you can edit with Paradise and choose your custom title for the selected folder.


Widget Label Colour

You can set a custom colour for each Widgets label separately. Just use 3D Touch to edit with Paradise and choose your custom colour for the selected widget label.


Widget Label Background Colour

You can set custom background colours for each Widgets label separately. Just use 3D Touch to Edit With Paradise and choose your custom colour for the selected widget label.


Widget Label Title

You can set a title for each Widgets separately. Just use 3D Touch to edit with Paradise and choose your custom title for the selected widget label.


Above customisation can be done for app icons, folders, widgets and with App Library folders too.


Batch Uninstalling Apps

You can use Paradise to uninstall multiple apps at a time it’s very easy and very fast.


Offloading App

You can use Paradise to offload multiple apps.


Appearance

Paradise can adapt your system setting dark or light mode, if you don’t want to use system appearance then you can colour everything to your liking so it will fit in with any set up.
