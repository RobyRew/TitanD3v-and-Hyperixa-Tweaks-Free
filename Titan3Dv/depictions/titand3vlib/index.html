TitanD3v Library

Description


Super simple universal library without any additional dependencies for TitanD3v tweaks.


Current Features:

• Custom cells

• Description cell

• Image cell

• Appearance picker

• Custom colour picker and system colour picker too (iOS 14 only)

• Image picker

• Date picker

• Custom enable & disable cell

• Grid cells

• Font picker (iOS 13 and above)

• AppList

• Bug report

• Drawer menu

• Changelog

• Backup & Restore

• Tweak list

• Crew

• Utilities manager


Compatibility




Supported iPhone on iOS 13 & 14
Unsupported for iPad
